# Redit

## Docker installation

### Requirements
- docker > 1.18
- docker-compose > 1.25.0

### Steps
- Clone the project source
  ```
   git clone https://github.com/tn-aixpa/redit.git
  ```

- Run from project root
  ```
   docker compose up
   ```

- Interface console is available at url
  ```
  http://localhost:8080
  ```
